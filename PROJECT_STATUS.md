## 已完成功能

### 1. 题库系统
- **总题数**：1500题（初/中/高各500题）
- **数据文件**：`scenarios.json`（JSON格式，从scenarios.js转换而来）
- **复杂度评分**：每道题含`complexity`字段（1-9分）
  - 初级：复杂度1-3分，翻前基础决策
  - 中级：复杂度4-6分，翻后中等思考量
  - 高级：复杂度7-9分，湿润牌面复杂计算
- **数据质量**：
  - 已修复action-reasoning不匹配问题（修复740道）
  - 每道题包含：id, difficulty, street, position, hand, board, optimalAction, reasoning, equity, potOdds, mdf, complexity等字段

### 2. 游戏核心
- 封面三模块：开始游戏、概率计算、复盘模式
- 难度选择：初级/中级/高级
- 游戏流程：发牌 → 决策 → 反馈 → 下一题
- 功能：实时胜率显示、EV计算、MDF计算

### 3. 概率计算教学模块
- **四步渐进式教学**：
  1. 简单理解：概念介绍（胜率、赔率、MDF）
  2. 简单使用：一键计算器（预设手牌胜率）
  3. 深入理解：数学原理+公式
  4. 深入使用：专业工具（手牌对抗计算器、MDF计算器）
- **牌型格式提示**：已添加输入格式说明（s=♠ h=♥ d=♦ c=♣）

### 4. 复盘模式（简化版）
- **五步骤流程**：翻前 → 翻牌 → 转牌 → 河牌 → 结束复盘
- **功能**：
  - 每街可输入牌面、底池、下注额、行动
  - 实时显示胜率、赔率、建议（简化估算）
  - 结束复盘可给复杂度评分（1-9）
  - 生成题目保存到localStorage，标注"复盘模式收录"
- **限制**：胜率用随机值估算，非精确计算

### 5. UI/UX优化
- 暗色主题（金色+深色背景）
- 修复下拉框在暗色主题下的显示问题
- 添加牌桌图片（assets/table.png）
- 响应式布局，支持移动端
