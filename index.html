什么是胜率、赔率、MDF？

🎯 胜率 (Equity)

你的牌平均能赢多少。比如AA对抗随机牌，胜率约85%。

例子：翻牌前AA vs KK，AA胜率约80%

💰 赔率 (Odds)

你需要投入多少才能赢得底池。赔率好=风险小、回报大。

例子：底池100，对手下注50，你的赔率是3:1（100:50+50）

🛡️ MDF (Minimum Defense Frequency)

面对下注，你最少要跟注或加注的频率，防止对手随意诈唬。

公式：MDF = 底池 / (底池 + 下注额)。如下注50到100底池，MDF = 100/(100+50) = 67%

数学原理详解

胜率计算原理

胜率 = (获胜组合数 / 总组合数) × 100%

例如：翻牌圈有2张补牌（Outs），剩余47张未知牌，胜率 ≈ (47-9)/47 ≈ 81%

快速估算：Outs × 2 = 翻牌圈胜率%，Outs × 4 = 转牌圈胜率%

赔率计算

Pot Odds = 底池大小 / 需要跟注的金额

Implied Odds = (底池 + 预期后续可赢取) / 需要跟注的金额

决策：如果胜率 > 1/(赔率+1)，则跟注有利可图

MDF详解

MDF (Minimum Defense Frequency) = 底池 / (底池 + 下注额)

作用：如果你防守频率低于MDF，对手可以任意下注诈唬获利。

实例：底池100，下注50，MDF = 100/(100+50) = 67%。你需要至少跟注2/3的牌。
